# Gestiones Laborales

Proyecto para replicar la página web de "Gestiones Laborales" con un diseño moderno y responsive.

## Estructura de Archivos
- `index.html`: Página principal.
- `services.html`: Página de servicios.
- `about.html`: Página "Sobre mí".
- `contact.html`: Página de contacto.

## Uso
1. Clona este repositorio.
2. Abre los archivos HTML en tu navegador.

## Zona privada con Laravel y Filament

Se incluye una aplicación Laravel en el directorio `private-zone` que provee un panel de administración usando Filament. Consulta [docs/PRIVATE_ZONE_SETUP.md](docs/PRIVATE_ZONE_SETUP.md) para los pasos de instalación y uso.
